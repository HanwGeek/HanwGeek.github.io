ChromeHeadless+MySQL模式爬虫
2018-07-09|Crawler

由于谷歌将Chrome从59版后推出了Headless无头浏览器模式，可用于搭配Selenium+Chromedriver实现爬虫。导致原先的常用PhantomJS直接停止维护，就此退出历史舞台。本文记录一下首次使用Chrome Headless爬虫的过程。

环境配置

本次爬虫基本环境为Ubuntu16.04 + Python3.5

Python爬虫模块

安装爬虫基本模块Selenium, BeautifulSoup4, Requests

Chrome

在谷歌为数不多能够访问的Chrome官网上，下载合适的版本

本机采用的版本是较新的v67。

Chromedriver

需要注意的是,下载的Chromedriver版本需要和Chrome版本相互对应。

本次采用从淘宝镜像源下载的Chromedriver v2.40（在镜像中有详细的版本对应说明note.txt )

MySQL

安装MySQL与Python接口模块PyMySQL
