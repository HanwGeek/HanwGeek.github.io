Gradient Boost

梯度提升方法(Gradient Boost)在基本的前向分布算法上进行命题的加强。

在之前我们所提到的前向分布算法中，假定损失函数为平方误差函数(或指数误差函数)，构造学习器以拟合残差。但实际上，假如我们的优化目标不仅仅是损失函数，一般情况下会加入正则项以限制模型的复杂度。在这种情况下，直接对优化目标进行优化难度较大。

因此，在构造基学习其拟合残差的主体思想不变的情况下，利用最速下降法的近似方法，以损失函数的负梯度，来代替当前的残差值，即:

要使得残差尽可能小，根据梯度下降法可知，需要其沿变量梯度方向下降。因此使得新增的基学习器去拟合损失函数在$f(x)$方向上的负梯度，以达到该目的。

Gradiet Boosting框架可以结合多种基学习器以在不同问题中达到最优效果。在实际情况下，梯度提升决策树(GBDT, Gradient Boosting Decision Tree)应用最为广泛，总体效果也叫好。

XGBoost

XGBoost(eXtreme Gradient Boosting)是基于Gradient Boosting的进一步改进。是目前在机器学习竞赛领域应用最多，最为有效的学习方法之一，由陈天奇大佬开发。

由于我们的目标是最小化损失函数，然而树的结构的可能情况太多，不能够一一构建并遍历一计算损失函数的最小值。

因此采用贪心方法，将树的叶节点逐步分裂。在叶节点分裂时，需要确定该处分裂是否可以使得损失函数进一步变小。采用以下算法：

  • 从深度为0的空树开始

  • 对于每个叶节点，枚举所有特征

  • 对于每个特征，将训练样本按该特征值进行排序
  • 从左到右遍历分割点位置，计算$Gain$值以确定最佳分割点

该式的意义是若按该处进行分割后，损失函数的下降量，即左子树右子树损失值和减去分割前损失函数和以及增加叶节点分割带来的树复杂度罚项(+1)。

值得注意的是构建单独一棵树的时间复杂度为$O(ndKlogn)$，其中$d$为特征树，$K$为树层数，$nlogn$为排序时间复杂度。同时在编程实现时可以采用更底层的存储优化或并行优化方法。因而，XGBoost方法可以应用在大型数据集上，实现了其论文题中的A Scalable Tree Boosting System。
